# Stough et al. 2008 — 90-Day Double-Blind Placebo-Controlled RCT

The **most important efficacy trial** in the [[cdri-08|CDRI-08]] / Mindwise evidence base. Single largest, longest, most directly cited.

## Citation

> Stough C, Downey LA, Lloyd J, Silber B, Redman S, Hutchison C, Wesnes K, Nathan PJ. (2008). Examining the Nootropic Effects of a special extract of *Bacopa monniera* on Human Cognitive Functioning: 90 day Double-Blind Placebo-Controlled Randomized Trial. *Phytother. Res.*, 22: 1629-1634.

## Study design

- **Format:** Double-blind, placebo-controlled, computer-generated randomization
- **Sponsor:** Keen Health Pty Ltd / Flordis Pty Ltd
- **Site:** Brain Sciences Institute, Swinburne University, Hawthorn, Victoria, Australia
- **Active product:** "Mindwise" — Standardized Bacopa monnieri extract (CDRI, Lucknow), no less than 55% combined bacosides — see [[55-percent-bacosides]]
- **Dose:** 2 × 150 mg capsules / day (= 300 mg/day total)
- **Duration:** 90 days
- **Participants:** 170 recruited, **62 completed**
- **Inclusion:** healthy adults 18–60, willing informed consent, non-smoker
- **Exclusion:** dementia history, psychiatric disorders, neuro/endocrine/GI/bleeding disorders, chronic illness, pregnancy/lactation, anticoagulants, antidepressants, antipsychotics, anxiolytics, ACE inhibitors, anti-Parkinson's, other cognitive enhancers

## Methodology

Cognitive Drug Research (CDR) Computerised Assessment System measured **5 cognitive factors**:
1. Secondary memory
2. Working memory
3. Speed of memory
4. Speed of attention
5. Accuracy of attention

Specific neuropsychological tasks:
- Word presentation, picture presentation, immediate word recognition
- Simple reaction time, digit vigilance, choice reaction time
- Spatial working memory, numeric working memory
- Delayed word recognition, delayed picture recognition
- **Rapid Visual Information Processing (RVIP)** — key task for false-positive measure

Weekly symptom tracking (positive: energy, arousal, cognitive clarity; negative: fatigue, nausea, etc.)

## Findings (the citable claims)

### Primary positive findings
- ✅ **Significant improvement in 'Working Memory' factor** — specifically **spatial working memory accuracy**
- ✅ **Reduced false-positives** in the Rapid Visual Information Processing task

### Secondary observations
- Several tasks not significantly improved
- **Positive tendency** toward improvement in attention, working memory, and psychomotor tasks (statistical analysis)
- Authors recommended larger sample sizes for future studies

## Quality rating
**Medium** — per the CTD assessment:
- Significant improvement in working memory + RVIP false positives
- Medium sample (62 completers)
- Larger number of variables (multiple-comparison concern)
- Low level of significance on some
- Lack of state anxiety reduction (contradicts 2001 study)

## What it supports for marketing

✅ "Mindwise improves working memory" (with the specific spatial working memory caveat)
✅ "Improves visual information processing accuracy"
✅ "90-day clinical trial" reference (supports the [[8-12-week-onset-claim|8-12 week onset]])
✅ "Mindwise" name explicitly used in study (the trial is technically branded)
✅ Same standardized 55% extract as the commercial product

❌ Don't overclaim "improves all memory" — the study was specific
❌ Don't claim acute / fast onset — this is a 90-day study by design
