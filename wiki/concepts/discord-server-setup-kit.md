# Discord Server Setup Kit

A fully-drafted Discord launch playbook that **already exists** in the repo at [racesims/kits/Discord_Server_Setup_Kit.html](../../racesims/kits/Discord_Server_Setup_Kit.html). It is the **single highest-leverage unblock** in the entire RaceSims [[90-day-execution-plan|90-day plan]] — ~30 minutes of execution closes the [[discord-community-gap|biggest community deficit vs VRH]].

## Status

- ✅ Kit drafted
- ❌ Server not yet launched
- ⏳ Waiting on: just doing it

## What the kit covers

Per the [[2026-03-25-vrh-battle-plan|VRH battle plan]] and the kit doc itself:

### Channel structure
- `#welcome`
- `#general-chat`
- `#setup-help`
- `#race-engineer-tips` ← **the killer differentiator** (see [[race-engineer-positioning]])
- `#deals-and-offers`
- `#customer-setups`
- `#race-events`
- `#buy-sell-trade`

### Programming
- **Weekly community races** (Sundays — match [[virtual-racing-hub|VRH]] cadence)
- **Post-race race-engineer commentary** — the asymmetric edge (VRH can't do this)
- **Weekly race-engineer-tips posts** from [[thambu]]
- **Discord-only discount codes** — gives people a reason to join AND stay
- **Product launch previews** — Discord first, then website (creates FOMO)

### Companion strategy
Don't only build from scratch — also partner with [[indian-racing-community|IRC]] (the largest existing Indian sim-racing Discord) for top-of-funnel reach.

## Existing infrastructure already in place

- ✅ [[racesims-bot|RaceSims Discord bot]] (`racesims/racesims-bot/`) — already coded, just needs server to run on
- ✅ Setup Kit doc — fully drafted
- ✅ [[race-engineer-positioning|Content angle]] — already defined
- ✅ Brand voice — already established

**The only thing missing is the click that creates the server.** This is genuinely a 30-minute task with massive downstream value.
