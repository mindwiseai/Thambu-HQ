# Morgan & Stevens 2010 — Older Persons Memory Trial

Clinical trial of [[cdri-08|standardized Bacopa monnieri extract]] in older persons. Single-dose study with notable limitations but adds to the elderly cohort evidence base.

## Citation

> Morgan A. and Stevens J. (2010). Does *Bacopa monnieri* improve memory performance in older persons? Results of a randomized, placebo-controlled, double blind trial. *J. Alternative & Complementary Medicine*, 16:753-759.

## Study design

- **Format:** Double-blind, placebo-controlled, randomized
- **Cohort:** Older / healthy older adults
- **Dose:** Single dose, standard concentration
- **Sample size:** 81
- **Dropout rate:** 20% (high)

## Findings

### Positive
- ✅ **Improved memory acquisition** demonstrated

### Null
- ❌ **No significant effects** for Bacopa compared to placebo (p > 0.05) on:
  - Visuospatial memory performance
  - Visuomotor skills

## Quality rating
**Medium** — per CTD assessment:
- Memory acquisition improvement demonstrated
- Null results on visuospatial memory and visuomotor skills
- Low sample size (81)
- High dropout rate (20%) — concerning for statistical power

## Significance for Mindwise

This is the **most cautious** trial in the elderly cohort evidence base. It demonstrates a real but narrow effect (memory acquisition specifically) and provides honest null results on adjacent dimensions. Its main value is **balance**:

- It prevents Mindwise from overstating the elderly evidence base
- It supports the specific "memory acquisition" claim
- It's intellectually honest — citing it alongside the more positive trials shows scientific integrity

## What it supports
✅ "Improves memory acquisition in older adults" (specific narrow claim)
✅ Adds to the elderly cohort evidence base alongside [[barbhaiya-2008-elderly-trial]] and [[calabrese-2008-elderly-trial]]

## What it does NOT support
❌ Visuospatial memory improvement
❌ Visuomotor / motor coordination claims
❌ Strong / definitive elderly cognition claims (the trio of elderly trials together support broader claims; this one alone is too cautious)

## Why it's still valuable

In a regulatory or doctor-facing conversation, **citing a null-result trial alongside positive ones builds trust**. The pharmaceutical industry's credibility issue is exactly the *suppression* of null results. Mindwise can position differently by openly disclosing its limited findings — that's part of the [[positioning-shift|"performance, not placebo wellness"]] integrity story.
